# 🕵️ Шпион — Spy Party Game

Многопользовательская игра "Шпион" на Node.js + Socket.io

## Установка и запуск

```bash
# 1. Установить зависимости
npm install

# 2. Запустить сервер
npm start
# или для разработки:
npm run dev

# 3. Открыть в браузере
http://localhost:3000
```

## Как играть

1. **Создать комнату** — введите имя и нажмите "Создать комнату"
2. **Поделиться кодом** — нажмите на код комнаты, чтобы скопировать
3. **Настроить игру** (хост):
   - Количество игроков (3–15)
   - Количество шпионов (1–5)
   - Время игры (1–30 минут)
   - Редактировать список мест
4. **Начать игру** — каждый получает свою роль приватно
5. **Шпион** не знает место, но видит список всех возможных мест
6. **Мирные** знают место, но не знают кто шпион
7. Игроки задают друг другу вопросы, пытаясь раскрыть шпиона
8. **Голосование** — выберите подозреваемого и нажмите "Голосовать"

## Технологии

- Node.js + Express (сервер)
- Socket.io (реальное время)
- Vanilla HTML/CSS/JS (клиент)
- Без базы данных (всё в памяти)
